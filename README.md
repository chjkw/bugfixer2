# bugfixer README

SW 오류 자동 수정 엔진 실행 및 결과 확인 그리고 패치기능이 포함된 VS Code 확장 프로그램입니다.

## 기능

다음 기능을 포함합니다.
- Saver 엔진 실행, 패치 생성 및 적용
- NPEX를 이용한 패치 생성 및 적용


## 준비물

- Saver 실행 준비
  - Virtual Box
  - Saver VM


## 제약사항

- Saver 관련 기능만 실행가능합니다.
- NPEX는 준비 중입니다.
